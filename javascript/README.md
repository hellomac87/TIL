## 계획

4주간 스쿨을 진행하면서, 온전히 혼자 생각하고 공부 할 수 있는 시간은 네시간 남짓 되는 것 같다.
귀가 후 따로 학습하는 시간을 제외한 시간을 잘 활용해야 한다.

1. 수업 내용 복습 (30분)
2. 수업 진행 또는 과제 알고리즘 문제 전부 풀어보기 (1시간)
3. 틱택토 게임 만들기 (1시간)
4. 책(러닝자바스크립트) 학습하기 (1시간)
5. 리액트 (30분)
